📌 Overview

This workshop provides a hands-on experience building and deploying cloud resources using Amazon Web Services (AWS). Participants will provision infrastructure, deploy application components, configure security, and follow AWS best practices.

The lab is designed to strengthen understanding of:

Cloud architecture fundamentals

AWS core services

Infrastructure provisioning

Security and IAM configuration

Resource cleanup and cost management

🎯 Learning Objectives

By the end of this workshop, you will be able to:

Understand AWS core services and their use cases

Deploy infrastructure using the AWS Management Console or CLI

Configure networking and access controls

Implement best practices for security and cost optimization

Clean up AWS resources safely to avoid unnecessary charges

🏗 Architecture Overview

Complete each module in order.

Module	Description

Cleanup	Remove all created resources
🔐 Security Best Practices

Use IAM roles instead of root credentials

Follow least privilege access principles

Avoid hardcoding credentials

Enable logging and monitoring

💰 Cost Management

⚠️ AWS resources may incur charges if not cleaned up.

Use AWS Free Tier resources when possible

Monitor usage in AWS Billing Dashboard

Delete all created resources after completing the lab

🧹 Cleanup Instructions

After completing the workshop:

Terminate EC2 instances

Delete S3 buckets

Remove CloudFormation stacks

Delete custom IAM roles/policies (if created)

Verify no running services remain in the AWS Console
